# DriveFlix

A personal Netflix-style streaming platform powered by Google Drive.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui
- React Router
- Axios
- TanStack Query

## Project Structure

frontend/
backend/
docs/

## Features

- Google Login
- Google Drive Streaming
- Netflix Style UI
- TV Friendly
- Continue Watching
- Search
- Favorites

## Run

npm install

npm run dev
